TinyURL with MEAN Stack

In this project, an URL shortening service is built with MEAN Stack(MongoDB, Express, AngularJS, Node.js). It is designed with a MVC framework with a REST API backend.

The source code can be found on my Github.

For the shortened url lookup page, statistics related to the url is shown, including the total number of clicks(hourly, monthly, yearly), referrer link, the geo-location of the visitor's, platform and brower information.

I have hosted my app on DigitalOcean .(This demo may not work, because unlike heroku, DigitalOcean charged a certain amount for hosting. So I may stop this host sometimes)
